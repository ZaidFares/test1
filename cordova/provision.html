To provision the trusted assets store to the browser, it needs to be loaded by the browser and stored in the browser's local storage:
  • Provide a name for the trusted assets store to be saved into the local storage. This will also be the name the application provides as the trusted assets store.
  • Select the trusted assets store file. The file will be automatically loaded and stored.
  • The contents of the trusted assets store will be displayed in the browser for verification if required.
